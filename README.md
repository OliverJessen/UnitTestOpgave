# UnitTestOpgave

Vi fik til opgave at printe ud byer ud fra en string[] array. Man kunne ikke printe ud en by hvis der var mindre end 2 karakterer, og hvis der var to karakterer skulle den printe ud navnet pÃ¥ byen/byerne. Hvis den havde karakteren "*", skulle den printe ud alle byer. hvis den havde et del af et ord, skulle den printe ud ordet.
